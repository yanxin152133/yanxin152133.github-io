less 命令

less 命令与more 命令类似，less 命令可以随意浏览问文件，而more仅能向前移动，却不能向后移动，而less 在查看之前不会加载整个文件。

语法

参数

  • -b <缓冲区大小> 设置缓冲区的大小
  • -e 当文件显示结束后，自动离开
  • -f 强迫打开特殊文件，例如外围设备代号、目录和二进制文件
  • -g 只标志最后搜索的关键词
  • -i 忽略搜索时的大小写
  • -m 显示类似more命令的百分比
  • -N 显示每行的行号
  • -o <文件名> 将less 输出的内容在指定文件中保存起来
  • -Q 不使用警告音
  • -s 显示连续空行为一行
  • -S 行过长时间将超出部分舍弃
  • -x <数字> 将”tab”键显示为规定的数字空格
  • /字符串：向下搜索”字符串”的功能
  • ?字符串：向上搜索”字符串”的功能
  • n：重复前一个搜索（与 / 或 ? 有关）
  • N：反向重复前一个搜索（与 / 或 ? 有关）
  • b 向后翻一页
  • d 向后翻半页
  • h 显示帮助界面
  • Q 退出less 命令
  • u 向前滚动半页
  • y 向前滚动一行
  • 空格键 滚动一页
  • 回车键 滚动一行
  • [pagedown]： 向下翻动一页
  • [pageup]： 向上翻动一页
