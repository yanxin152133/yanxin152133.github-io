cd 命令

cd 命令用于切换当前工作目录值dirName（目录参数）。
其中dirName表示法可谓绝对路径或相对路径。若目录名称省略，则变换至使用者的home目录（也就是刚login时所在的目录）。
另外，“~”也表示为home目录的意思，“.”则是表示目前所在的目录，“..”则表示目前目录位置的上一层目录。

语法

  • dirName：要切换的目标目录。
