repquota 命令

repquota 命令用于检查磁盘空间限制的状态。
执行repquota指令，可报告磁盘空间限制的状况，清楚得知美味用户或每个群组已使用多少空间。

语法

参数

  • -a 列出在/etc/fstab文件里，有加入quota设置的分区的使用状况，包括用户和群组。
  • -g 列出所有群组的磁盘空间限制。
  • -u 列出所有用户的磁盘空间限制。
  • -v 显示该用户或群组的所有空间限制。
