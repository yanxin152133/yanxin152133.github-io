lpr 命令

lpr 命令用来将一个或多个文件放入打印队列等待打印。
lpr可以用来将料资送给本地或是远端主机来处理。

语法

参数

  • -p Printer: 将资料送至指定的打印机 Printer，预设值为 lp。
