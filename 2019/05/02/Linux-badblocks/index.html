badblocks 命令

badblocks 命令用于检查磁盘装置中损坏的区块。
执行指令时须指定所要检查的磁盘装置，及此装置的磁盘区块数。

语法

参数

  • -b<区块大小> 指定磁盘的区块大小，单位为字节。
  • -o<输出文件> 将检查的结果写入指定的输出文件。
  • -s 在检查时显示进度。
  • -v 执行时显示详细的信息。
  • -w 在检查时，执行写入测试。
  • [磁盘装置] 指定要检查的磁盘装置。
  • [磁盘区块数] 指定磁盘装置的区块总数。
  • [启始区块] 指定要从哪个区块开始检查。
