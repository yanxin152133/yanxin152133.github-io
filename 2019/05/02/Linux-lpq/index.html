lpq 命令

lpq 命令用于查看一个打印队列的状态，该程序可以查看打印机队列状态及其所包含的打印任务。

语法

参数

  • -P 指定一个打印机，否则使用默认打印机或环境变量PRINTER指定的打印机。
  • -l 打印组成作业的所有文件的信息。
