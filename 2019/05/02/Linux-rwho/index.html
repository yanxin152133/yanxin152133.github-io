rwho 命令

rwho 命令用于查看系统用户。

rwho指令的效果类似who指令，但它会显示局域网里所有主机的用户。主机必须提供rwhod常驻服务的功能，方可使用rwho指令。

语法

参数

  • -a 　列出所有的用户，包括闲置时间超过1个小时以上的用户。
