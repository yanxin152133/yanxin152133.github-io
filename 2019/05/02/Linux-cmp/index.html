cmp 命令

cmp 命令是用于比较两个文件是否有差异。
当相互比较的两个文件完全一样的时候，则该命令不会显示任何信息。若有差异，预设会标示出第一个不同之处的字符和列数编号。若不指定任何文件名称或是所给予的文件名为”-“,则cmp命令会从标准输入设备读取数据。

语法

参数

  • -c或–print-chars 　除了标明差异处的十进制字码之外，一并显示该字符所对应字符。
  • -i<字符数目>或–ignore-initial=<字符数目> 　指定一个数目。
  • -l或–verbose 　标示出所有不一样的地方。
  • -s或–quiet或–silent 　不显示错误信息。
  • -v或–version 　显示版本信息。
  • –help 　在线帮助。
