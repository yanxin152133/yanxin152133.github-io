mattrib 命令

mattrib 命令用来变更或显示MS-DOS 文件的属性。
mattrib 为mtools工具指令，模拟MS-DOS的attrib指令，可变更MS-DOS文件的属性。

语法

参数

  • -a/+a 除去/设定备份属性。
  • -h/+h 除去/设定隐藏属性。
  • -r/+r 除去/设定唯读属性。
  • -s/+s 除去/设定系统属性。
  • -/ 递回的处理包含所有子目录下的档案。
  • -X 以较短的格式输出结果。
