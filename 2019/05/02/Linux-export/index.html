export 命令

export 命令用于设置或显示环境变量。

在shell中执行程序时，shell会提供一组环境变量。export可新增，修改或删除环境变量，供后续执行的程序使用。export的效力仅及于该次登录操作。

语法

参数

  • -f 　代表[变量名称]中为函数名称。
  • -n 　删除指定的变量。变量实际上并未删除，只是不会输出到后续指令的执行环境中。
  • -p 　列出所有的shell赋予程序的环境变量。
