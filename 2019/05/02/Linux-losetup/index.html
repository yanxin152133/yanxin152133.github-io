losetup 命令

losetup 命令用于设置循环设备。
循环设备可把文件虚拟成区块设备，籍以模拟整个文件系统，让用户得以将其视为硬盘驱动器、光驱或软驱等设备，并挂入当作目录来使用。

语法

参数

  • -d 卸除设备。
  • -e<加密方式> 启动加密编码。
  • -o<平移数目> 设置数据平移的数目。
