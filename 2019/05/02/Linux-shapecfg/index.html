shapecfg 命令

shapecfg 命令用于管制网络设备的流量。
自Linux-2.15开始，便支持流量管制的功能 。

语法

参数

  • attach 将流量管制器与实际的网络设备结合。
  • speed 设置流量管制器的对外传输带宽。
