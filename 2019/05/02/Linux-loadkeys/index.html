loadkeys 命令

loadkeys 命令可以根据一个键盘定义表改变 linux 键盘驱动程序转译键盘输入过程。详细的说明请参考 dumpkeys。

语法

参数

  • -v –verbose：印出详细的资料，你可以重复以增加详细度。
  • -q –quiet：不要显示任何讯息。
  • -c –clearcompose：清除所有 composite 定义。
  • -s –clearstrings：将定串定义表清除。
