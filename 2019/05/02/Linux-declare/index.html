declare 命令

declare 命令用于声明shell变量。

declare为shell指令，在第一种语法中可用来声明变量并设置变量的属性([rix]即为变量的属性），在第二种语法中可用来显示shell函数。若不加上任何参数，则会显示全部的shell变量与函数(与执行set指令的效果相同)。

语法

参数

  • +/- 　“-“可用来指定变量的属性，”+”则是取消变量所设的属性。
  • -f 　仅显示函数。
  • r 　将变量设置为只读。
  • x 　指定的变量会成为环境变量，可供shell以外的程序来使用。
  • i 　[设置值]可以是数值，字符串或运算式。
