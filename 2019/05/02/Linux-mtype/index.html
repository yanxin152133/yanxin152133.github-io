mtype 命令

mtype 命令为mtools工具指令，模拟MS-DOS的type指令，可显示MS-DOS文件的内容。

语法

参数

  • -s 去除8位字符码集的第一个位，使它兼容于7位的ASCII。
  • -t 将MS-DOS文本文件中的”换行+光标移至行首”字符转换成Linux的换行字符。
