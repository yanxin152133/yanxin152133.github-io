write 命令

write 命令用于传讯息给其他使用者。
使用权限：所有使用者。

语法

参数

  • user : 预备传讯息的使用者帐号
  • ttyname : 如果使用者同时有两个以上的 tty 连线，可以自行选择合适的 tty 传讯息
