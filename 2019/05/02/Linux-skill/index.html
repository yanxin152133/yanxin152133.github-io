skill 命令

skill 命令送个讯号给正在执行的程序，预设的讯息为 TERM (中断)，较常使用的讯息为 HUP、INT、KILL、STOP、CONT 和 0。

讯息有三种写法：分别为 -9、-SIGKILL、-KILL，可以使用 -l 或 -L 已列出可使用的讯息。
使用权限：所有使用者。
其他相关的命令：kill
s

语法

一般参数

  • -f 快速模式/尚未完成
  • -i 互动模式/ 每个动作将要被确认
  • -v 详细输出/ 列出所选择程序的资讯
  • -w 智能警告讯息/ 尚未完成
  • -n 没有动作/ 显示程序代号

参数

选择程序的规则可以是：终端机代号、使用者名称、程序代号、命令名称。

  • -t 终端机代号 ( tty 或 pty )
  • -u 使用者名称
  • -p 程序代号 ( pid )
  • -c 命令名称可使用的讯号
