升级npm 命令

sudo npm install npm -g

升级node.js命令步骤

sudo npm cache clean -f #----- 先清除 npm cache
npm install -g n #----- 安装 n 模块
sudo n stable #----- 安装最新的 node.js 稳定版

查看版本

node -v    # 查看node.js版本
npm -v     # 查看npm版本
